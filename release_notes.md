## Jellyfish 0.0.1

Erste öffentliche Version.

### Features
- Textkürzel mit automatischer Expansion
- Ordner für Textbausteine
- Datum-Platzhalter (`{TT.MM.JJJJ}`, `{HH:MM}` u. a.)
- Dropdown-Platzhalter (`{AUSWAHL:Option1|Option2|Option3}`) mit Vorschaufenster
- Auto-Update via GitHub Releases
- Verteilung als DMG
